# picoyplaca_predictor
An Java library to predict Quito's "pico y placa" restriction

Easy to use just execute the ExeApp.java

**Example 1:**</br>
Input the plate as text</br>
PCC-4321</br>
Input date as text in format dd/mm/yyyy</br>
08/01/2018</br>
Input time as text in format HH:MM</br>
08:19</br>
Please wait...</br>
Whops! You can't drive at this moment. Sorry :( </br>
